# 开发首页

首页能够展示带有分页的文章、全量的标签和全量的分类。

# 创建路由

在 routes/web.php 中移除之前的首页路由，并使用控制器模式。

//Route::get('/', function () {
//    return view('welcome');
//});

Route::get('/', 'IndexController@index');

# 创建控制器

执行以下命令创建首页控制器:

php artisan make:controller IndexController

编辑 app/Http/Controllers/IndexController.php，添加 index 方法:

public function index()
{
    return view('index.index');
}

# 创建视图

创建 resources/views/index/index.blade.php 文件，随意填入一些文字，在浏览器中访问项目首页，如果能看到刚刚填入的内容则说明路由和控制器已经联通。

# 取数据并进行渲染

完善 IndexController 的 index 方法:

public function index()
{
    $posts = Post::with(['user', 'category', 'tags'])->paginate();
    $categories = Category::all();
    $tags = Tag::all();

    return view('index.index', compact('posts', 'categories', 'tags'));
}

完善 index.blade.php:

<h1>Posts</h1>
<ul>
  @foreach($posts as $post)
    <li>
      <span style="margin-right: 12px; color: rosybrown">{{ $post->category->name }}</span>
      <a>{{ $post->title }}</a>

    </li>
  @endforeach
</ul>

<br>

<h1>Categories</h1>
<div>
  @foreach($categories as $category)
    <span style="margin-right: 12px; color: rosybrown">{{ $category->name }}</span>
  @endforeach
</div>

<br>

<h1>Tags</h1>
<div>
  @foreach($tags as $tag)
    <span style="margin-right: 12px; color: rosybrown">{{ $tag->name }}</span>
  @endforeach
</div>
