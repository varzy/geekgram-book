# 开发环境

开发环境如何搭建并不在本书讨论范围内，毕竟一万个人有一万套开发环境。但不管用什么开发环境，至少需要保证有下面几个软件:

  • PHP
  • Node
  • Composer
  • Npm
  • Git

# Windows

对于 Windows 平台，我推荐使用 Laragon 搭建一站式的开发环境。同时，你应当使用 bash 终端来执行命令，因为本书中所有的命令都是 linux like 的。
